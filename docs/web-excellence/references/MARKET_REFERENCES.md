# Sites de Referência por Categoria de Mercado

## Visão Geral

Referências de mercado servem como benchmark de qualidade. Este documento cataloga os melhores exemplos por vertical, analisando o que torna cada site excelente, padrões de design utilizados, e abordagem técnica. Atualizado para refletir tendências de 2025-2026.

---

## 1. SaaS

### 1.1 Linear (linear.app)

| Aspecto | Análise |
|---------|---------|
| **Por que é excelente** | Benchmark de dark mode, motion design, e performance percebida |
| **Design patterns** | Minimal hero, feature sections com scroll animations, changelog público |
| **Motion** | Spring physics, GPU-only transforms, stagger reveals |
| **Tipografia** | Inter, hierarquia limpa, fluid sizing |
| **Cores** | Dark mode primary, gradientes sutis, accent roxo |
| **Performance** | SSG + ISR, imagens otimizadas, lazy loading agressivo |
| **Lição** | Craft e atenção ao detalhe em cada micro-interação elevam a percepção de qualidade |

### 1.2 Vercel (vercel.com)

| Aspecto | Análise |
|---------|---------|
| **Por que é excelente** | Minimalismo extremo, performance perfeita (CWV verdes), developer marketing |
| **Design patterns** | Split hero, code-first demos, performance benchmarks visíveis |
| **Motion** | Sutil, funcional, nada gratuito — cada animação comunica algo |
| **Tipografia** | Geist (custom), monospace para code, hierarquia clara |
| **Cores** | Preto/branco com accent mínimo, gradientes preto→cinza escuro |
| **Performance** | Edge rendering, ISR, imagens AVIF, perfect Lighthouse |
| **Lição** | Menos é mais — confiança na simplicidade quando o produto é forte |

### 1.3 Stripe (stripe.com)

| Aspecto | Análise |
|---------|---------|
| **Por que é excelente** | Clareza de comunicação, developer docs padrão-ouro, design refinado |
| **Design patterns** | Animated gradients, interactive code samples, tabbed product sections |
| **Motion** | Gradient mesh backgrounds, hover reveals, smooth scrolling |
| **Tipografia** | Custom font, excelente hierarquia, muitos pesos |
| **Cores** | Gradientes vibrantes sobre fundo escuro/claro, accent blue |
| **Performance** | Edge caching, critical CSS, preload strategy |
| **Lição** | É possível ser visualmente impactante E altamente informativo |

### 1.4 Notion (notion.so)

| Aspecto | Análise |
|---------|---------|
| **Por que é excelente** | Whitespace, illustrações carismáticas, flexibilidade de uso demonstrada |
| **Design patterns** | Product-led hero, use case sections, template gallery |
| **Motion** | Ilustrações animadas, parallax suave |
| **Tipografia** | Rund Display (custom headings), limpa e amigável |
| **Cores** | Fundo off-white, accent preto + cores de produto |
| **Performance** | SSG para marketing, ISR para templates |
| **Lição** | Illustrações com personalidade diferenciam de screenshots genéricos |

## 2. E-commerce

### 2.1 Apple (apple.com)

| Aspecto | Análise |
|---------|---------|
| **Por que é excelente** | Pioneira do scroll storytelling, product photography como arte |
| **Design patterns** | Full-screen hero, scroll-driven animations, bento grid |
| **Motion** | Scroll-linked animations com IntersectionObserver + requestAnimationFrame |
| **Tipografia** | SF Pro, tamanhos enormes, pouco texto por viewport |
| **Cores** | Fundo preto para destaque de produto, branco para specs |
| **Performance** | Lazy loading de vídeos, responsive images com art direction |
| **Lição** | Storytelling visual com scroll cria experiência imersiva de produto |

### 2.2 Nike (nike.com)

| Aspecto | Análise |
|---------|---------|
| **Por que é excelente** | Editorial-meets-commerce, storytelling de marca, mobile-first |
| **Design patterns** | Editorial hero, grid de produtos, storytelling sections |
| **Motion** | Hover effects em produtos, scroll parallax, video backgrounds |
| **Tipografia** | Futura condensed (brand), bold headlines |
| **Cores** | Preto/branco dominante, cor do produto como accent |
| **Performance** | CDN global, responsive images, service worker |
| **Lição** | E-commerce pode ser editorial — storytelling vende mais que catálogo |

### 2.3 Glossier (glossier.com)

| Aspecto | Análise |
|---------|---------|
| **Por que é excelente** | UGC integration, community-driven, estética millennial/Gen-Z |
| **Design patterns** | UGC gallery, "shop the look", reviews integrados |
| **Motion** | Minimal, foco em fotografia |
| **Tipografia** | Clean sans-serif, rosa brand como accent |
| **Cores** | Rosa pastel (#FE7AB6), off-white, skin tones |
| **Performance** | JAMstack, CDN, image optimization |
| **Lição** | Social proof visual (UGC) é mais poderoso que fotografia profissional para DTC |

## 3. Portfolio / Agências

### 3.1 Awwwards Winners 2025

| Site | Destaque | Lição |
|------|----------|-------|
| **Lusion** | WebGL + 3D interactions | Tecnologia como arte quando bem executada |
| **Locomotive** | Smooth scroll + GSAP | Motion design premium eleva credibilidade |
| **Active Theory** | Experiências imersivas | Experiência > informação para agências criativas |
| **Resn** | Criatividade experimental | Portfolio como demonstração de capacidade |
| **Dogstudio** | Narrativa + técnica | Storytelling + craft como diferencial |

### 3.2 Padrões de Portfolio

| Padrão | Frequência | Quando Funciona |
|--------|-----------|----------------|
| Grid de projetos com hover preview | 80% | Portfolio com 5+ projetos |
| Case study pages individuais | 90% | Projetos complexos |
| Hero statement (não hero image) | 60% | Agências com posicionamento forte |
| Cursor customizado | 40% | Agências criativas (cuidado com acessibilidade!) |
| Dark mode predominante | 65% | Trabalho visual/criativo |

## 4. Restaurante / Food

### 4.1 Blue Bottle Coffee (bluebottlecoffee.com)

| Aspecto | Análise |
|---------|---------|
| **Por que é excelente** | Fotografia aspiracional, simplicidade japonesa, brand clarity |
| **Design patterns** | Full-bleed photography, minimal nav, editorial content |
| **Lição** | Fotografia de qualidade substitui design complexo no food vertical |

### 4.2 Sweetgreen (sweetgreen.com)

| Aspecto | Análise |
|---------|---------|
| **Por que é excelente** | Fresco, colorido, mobile-first ordering |
| **Design patterns** | Menu visual com fotos, location-based, seasonal highlights |
| **Lição** | Para food delivery, a imagem do prato É o CTA |

### 4.3 Padrões Food/Restaurant

| Padrão | Impacto |
|--------|---------|
| Hero com foto de prato principal | +25% engagement |
| Menu visual (fotos em vez de lista) | +30% pedidos |
| Localização/Horário visível above-fold | -20% bounce (usuário encontra info rápido) |
| Botão de reserva/pedido sticky | +15% conversão |
| Stories/conteúdo sazonal | +20% retorno |
